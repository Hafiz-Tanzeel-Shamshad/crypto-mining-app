# Crypto Mining App

A MERN stack app with MetaMask authentication and a mining dashboard.

## Folder Structure
- `frontend/` - React frontend
- `backend/` - Node.js/Express backend

## Getting Started

### Backend
```bash
cd backend
npm install
node server.js
```

### Frontend 
```bash
cd frontend
npm install
npm run dev
```

REACT_APP_API_URL=http://localhost:5000
